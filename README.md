## 💡 CSSによる 2.5D「抽選箱」の仕組み

このプロジェクトでは、JavaScriptによる複雑な描画やWebGL・CanvasAPI等を使わず、CSSの `z-index` と `position: absolute` だけで**「手が箱の中に潜り込む（箱の奥に消える）」**という空間的な魔法を実現しています。

### レイヤー（Z-index）階層設計
以下の3層構造（サンドイッチ状態）によって成り立っています。

1. **【奥側のレイヤー】（z-index: 10）** -> `.box-base`
   - 箱の天面の「ベースとなる赤い面」と、その上に乗る「暗闇の穴（黒いスリット）」を描画します。
   - この階層が一番奥の壁になります。

2. **【中間のレイヤー】（z-index: 15）** -> `.hand-container`
   - 手の絵文字（✋・✊）と、結果の書かれたくじ紙を配置します。
   - 上から下（穴へと）降りていき、奥側のレイヤーより手前を通過します。
   
3. **【手前側のレイヤー】（z-index: 20）** -> `.box-front-cover`
   - 箱の「正面の壁」と、「穴より手前のわずかなフチ（lip）」を描画します。
   - このレイヤーが **中間レイヤの上（前）** に立ちはだかります。
   
**👉 この構造により起きる錯覚**
手の要素がフチよりも下（奥）に下回ると、手前側の壁（z-index: 20）に物理的に隠れて見えなくなります。
これにより、「奥の天面の穴スリットと、手前側の壁の隙間を縫って、箱の奥側に入っていく」ように視覚的に誘導されています。

## ⏱️ アニメーションタイミング (`style.css` ＆ `script.js`)

一回のプロセスは約2.5秒です。
- `0% 〜 30%` (0 〜 0.75秒) : 手ぶらの手（✋）が上空から箱の中へ入る。
- `30% 〜 40%` (0.75秒 〜 1.0秒) : 手が箱の底で止まる。
- `40% 〜 70%` (1.0秒 〜 1.75秒) : 箱が揺れる（探っている）。
  - **JSで連動**：1.5秒経過したタイミングで、手の絵文字を「✋」から「✊（掴む）」に切り替え。
- `70% 〜 100%` (1.75秒 〜 2.5秒) : ✊の手がくじを引き上げる。
  - **CSSで連動**：引き始めの 70% の瞬間に、今まで透明（opacity:0）だったくじ紙の `opacity` を1にし、紙を初めて引き出したように見せる。
  - 2.5秒後、完全に箱の上空へ引き上げられた位置で「ポーン」と結果の文字が書かれた紙が開いて表示される仕組みです。
