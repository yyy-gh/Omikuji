# プロジェクト憲法 (Omikuji App)

## 1. 概要
日本の神社らしさを感じる、赤と白を基調としたおみくじアプリ。

## 2. 目的・利用シーン
- 自分用の毎日の運試しとして使用する。

## 3. ターゲット
- 自分自身

## 4. アプリの構成 / MVP（最小構成）の範囲
- Webブラウザ上で動作する。
- 「おみくじを引く」ためのボタンを設置する。
- ボタン押下後、『大吉』『中吉』『凶』のいずれかをランダムで画面に表示する。
- **デザイン**: 日本の神社風の雰囲気。赤と白を中心とした配色による洗練されたデザイン。

## 5. 技術スタック
- **構造**: HTML
- **スタイリング**: Vanilla CSS
- **ロジック**: Vanilla JavaScript
